### 日本の新書レーベルの状況

**岩波新書、中公新書、ちくま新書、講談社現代新書:**
- 2024年出版の新書: **ほとんど登録されていない**
- 検索結果に児童書や一般書籍は出るが、新書シリーズは含まれない

## 結論

**Google Books APIのデータベースには、日本の新書の登録に大幅な遅延があります。**

- 2007-2022年に出版された新書: 登録されている
- 2024-2025年に発見された新刊: **ほとんど登録されていない**
- 出版日データの信頼性: **低い**（未来の日付が多数）

**日本の主要新書レーベルは、Google Books APIでほとんど更新されていません。**

新刊の登録までに**数ヶ月から数年**かかる可能性があり、特に新書シリーズは優先度が低いと思われます。

## 取得可能な情報

Google Books APIから以下の情報が取得できました：

- タイトル
- 著者名
- 出版日
- ページ数（一部）
- 言語
- カテゴリ
- 説明文
- プレビュー可否

## テストスクリプト

### test-google-books.js
データベースに登録された最新の新書でテストします。

```bash
node gba/test-google-books.js
```

### test-known-books.js
既知の（古い）新書でテストします。

```bash
node gba/test-known-books.js
```

## 制限事項

- **レート制限**: テストではリクエスト間に1秒の遅延を設定
- **APIキー**: 現在は未使用（匿名リクエスト）
- **新刊の遅延**: 最新の新書は登録されていない可能性が高い

## 次のステップ

Google Books APIをこのプロジェクトで使用する場合：

1. **補完情報として利用** - openBDにない情報（レビュー、カテゴリ）を取得
2. **古い書籍の補完** - データベースの既存書籍に情報を追加
3. **APIキーの取得** - レート制限を緩和するため

ただし、**新刊検出にはopenBD APIの方が圧倒的に優れています**。
